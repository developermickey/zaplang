A modern programming language that compiles to JavaScript and native binaries. Simple like Python, fast like Go, built for full-stack web apps.

npm install -g zap-language
Get Started → See Examples 
// hello.zap
let name = "World"
print(`Hello, {name}!`)

fn greet(person) {
  return `Welcome, {person}!`
}

print(greet("Mukesh"))

// Loop from 1 to 5
loop i from 1 to 6 {
  print(`Count: {i}`)
}
$ zap run hello.zap
// server.zap
server on 3000 {

  get "/" {
    return "Welcome to Zap!"
  }

  get "/hello/:name" {
    return `Hello, {req.params.name}!`
  }

  post "/api/data" {
    let body = req.body
    return { ok: true, data: body }
  }

}
$ zap run server.zap
// native.zp — compiles to a real binary
fn fibonacci(n) {
  if (n <= 1) { return n }
  return fibonacci(n - 1) + fibonacci(n - 2)
}

let nums = [10, 20, 30, 40, 50]
print(nums.reverse())
print(fibonacci(10))
$ zap build native.zp --native -o myapp

Why Zap?

⚡

Blazing Fast

Compiles to optimized C99 for native speed, or runs on Node.js for instant startup.

🎯

Simple Syntax

Reads like plain English. No semicolons, no boilerplate — just write logic.

🌐

Full-Stack Ready

Built-in server block handles HTTP routes. Build APIs in minutes.

📦

Rich Stdlib

8 built-in modules: fs, http, os, math, time, crypto, str, arr — batteries included.

🔒

Secure by Default

Crypto module built-in. UUID, hashing, HMAC, and safe random — no extra packages.

🛠️

VS Code Extension

Syntax highlighting, snippets, hover docs, and auto-complete for .zap and .zp files.

Quick Start

1

Install

npm install -g zap-language
2

Create a file

echo 'print("Hello Zap!")' > hello.zap
3

Run it

zap run hello.zap
4

Build native binary

zap build hello.zap --native
